# Introduction

KandyPhonegap is a Phoengap plugin that makes Kandy services available on iOS and Android. It helps and this is based on Kandy native mobile SDK. It helps you to build cross-platform mobile apps using HTML5, Javascript and CSS. Wrap your app with Kandy PhoneGap plugin and deploy to mobile platforms.

This plugin encapsulates the Kandy mobile SDK. Developers could quickly embed collaboration widgets like video call, chat widget into their applications by declaring tags with predefined attribute names.
